## ISSUE FOUND AND FIXED

**Error:** ERROR: 42703: column "geofence_radius_meters" does not exist

### The Problem
File 03 was trying to add a CHECK constraint on a column that doesn't exist in the rooms table:
- Column: geofence_radius_meters
- Table: rooms
- Issue: Only the BUILDINGS table has this column, not rooms!

### The Solution Applied
**REMOVED** (Line 92):
```sql
ALTER TABLE rooms 
ADD CONSTRAINT check_geofence_radius CHECK (geofence_radius_meters > 0);
```

**ADDED** (Instead):
```sql
-- Ensure room capacity is positive
ALTER TABLE rooms 
ADD CONSTRAINT check_room_capacity CHECK (capacity > 0);

-- Ensure baseline pressure is in reasonable range
ALTER TABLE rooms 
ADD CONSTRAINT check_baseline_pressure CHECK (baseline_pressure_hpa > 0);
```

**KEPT** (Correct one):
```sql
ALTER TABLE buildings 
ADD CONSTRAINT check_building_geofence_radius CHECK (geofence_radius_meters > 0);
```
